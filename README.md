hacspec is the functional subset of Rust that can be used, together with a hacspec
standard library, to write succinct, executable, and verifiable specifications in
Rust.
These specifications can be translated into formal languages with [hax](https://hax.cryspen.com/).

There is a wealth of [specifications](https://github.com/hacspec/specs) of cryptographic algorithms and protocols to get you started.
